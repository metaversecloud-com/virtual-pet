# Virtual Pet 🐶🐕🐕‍🦺

Virtual Pet is a captivating mini-game developed using React and Node.js. The game allows users to choose, feed, train, and play with a virtual pet, gaining experience and leveling up as they interact with their pet.

## Features

- **Pet Selection:** Users can select from a variety of pets within the application.
- **Pet Care:** Daily activities such as feeding, playing, and training are essential for pet growth and happiness.
- **Growth System:** Pets evolve in stages from baby to adult, with visual changes to represent their growth.
- **Persistence:** User progress is saved, allowing for continuous interaction and care over time.
- **Social Interaction:** Users can see others' pets in the zone, fostering a community environment.

## Requirements

We are using Node version 18 or more

## Getting Started 🚀

Make sure to fill all environment variables. The model is in .env-example file

### Production Mode

1. Install dependencies

   ```
   npm install
   ```

2. Start the server
   ```
   npm start
   ```

### Development Mode

1. Start Server

   ```
   npm start
   ```

2. Start Client

   ```
   cd client && npm start
   ```

3. Visit `http://localhost:3001` and start taking care of your pet.
